# 个人设备部署方案

## 一、Docker Compose 一键部署

### 1.1 目录结构

```
memory-rag/
├── docker-compose.yml
├── Dockerfile
├── config.yaml
└── browser-extension/
    ├── manifest.json
    ├── content-script.js
    └── background.js
```

### 1.2 Docker Compose 配置

```yaml
version: '3.8'

services:
  postgres:
    image: pgvector/pgvector:pg16
    environment:
      POSTGRES_DB: memory_rag
      POSTGRES_USER: postgres
      POSTGRES_PASSWORD: ${DB_PASSWORD:-postgres}
    volumes:
      - pgdata:/var/lib/postgresql/data
      - ./init.sql:/docker-entrypoint-initdb.d/init.sql
    ports:
      - "5432:5432"
    healthcheck:
      test: ["CMD-SHELL", "pg_isready -U postgres"]
      interval: 5s
      timeout: 5s
      retries: 5

  memory-rag:
    build: .
    depends_on:
      postgres:
        condition: service_healthy
    environment:
      DB_HOST: postgres
      DB_PORT: 5432
      DB_NAME: memory_rag
      DB_USER: postgres
      DB_PASSWORD: ${DB_PASSWORD:-postgres}
      OPENAI_API_KEY: ${OPENAI_API_KEY}
    ports:
      - "8080:8080"  # HTTP API
    volumes:
      - ./config.yaml:/app/config.yaml
    command: ["serve", "--http", "--mcp"]

volumes:
  pgdata:
```

### 1.3 Dockerfile

```dockerfile
FROM golang:1.22-alpine AS builder

WORKDIR /app
COPY go.mod go.sum ./
RUN go mod download

COPY . .
RUN CGO_ENABLED=0 GOOS=linux go build -o memory-rag ./cmd/server

FROM alpine:latest
RUN apk --no-cache add ca-certificates
WORKDIR /root/

COPY --from=builder /app/memory-rag .
COPY config.yaml .

EXPOSE 8080
ENTRYPOINT ["./memory-rag"]
```

### 1.4 一键启动

```bash
# 1. 克隆项目
git clone https://github.com/your-repo/memory-rag.git
cd memory-rag

# 2. 配置环境变量
cat > .env <<EOF
OPENAI_API_KEY=sk-xxx
DB_PASSWORD=your_secure_password
EOF

# 3. 启动服务
docker-compose up -d

# 4. 验证
curl http://localhost:8080/api/stats
```

## 二、浏览器插件安装

### 2.1 Chrome/Edge 安装

1. 打开 `chrome://extensions/`
2. 开启"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `browser-extension` 目录

### 2.2 插件配置

```json
// manifest.json
{
  "manifest_version": 3,
  "name": "Memory RAG Assistant",
  "version": "1.0.0",
  "description": "捕捉 AI 对话并存储到本地知识库",
  "permissions": [
    "storage",
    "activeTab"
  ],
  "host_permissions": [
    "http://localhost:8080/*",
    "https://chatgpt.com/*",
    "https://claude.ai/*",
    "https://gemini.google.com/*"
  ],
  "content_scripts": [
    {
      "matches": [
        "https://chatgpt.com/*",
        "https://claude.ai/*",
        "https://gemini.google.com/*"
      ],
      "js": ["content-script.js"],
      "run_at": "document_end"
    }
  ],
  "background": {
    "service_worker": "background.js"
  },
  "action": {
    "default_popup": "popup.html"
  }
}
```
